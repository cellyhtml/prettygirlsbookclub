# Pretty Girls Book Club 🎀📚

Site oficial do **Pretty Girls Book Club**. Um espaço secreto e acolhedor onde páginas viram conexões, teorias e os melhores surtos literários!

## ✨ Estética
O site foi desenvolvido com uma estética **Editorial & Scrapbook**, trazendo o charme de recortes de jornal, polaroides, anotações à mão e colagens para o ambiente digital.

## 🚀 Funcionalidades
- **Biblioteca Digital:** Acervo completo de leituras passadas e futuras com filtros por categoria e downloads de PDFs.
- **Diário do Clube:** Registros detalhados de encontros, fotos (moodboards) e planejamentos mensais.
- **Resenhas Editoriais:** Páginas dedicadas aos livros do mês com resumos sem spoilers e ficha técnica.
- **Acessibilidade:** 
  - Menu flutuante para ajuste de tamanho de fonte e modo de Alto Contraste.
  - Integração com **VLibras** para tradução em Libras.
- **Responsividade:** Totalmente otimizado para celulares, tablets e desktops.

## 🛠️ Tecnologias
- **React** (Frontend)
- **Vite** (Build Tool)
- **Lucide React** (Ícones)
- **CSS Vanilla** (Estilização personalizada)
